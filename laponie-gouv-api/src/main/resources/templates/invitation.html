Vous avez été invité à rejoindre un groupe famille !

Bienvenue sur le site de gestion des listes de Noël.

Vous avez été invité dans un groupe famille, cela veut dire qu'une fois connectés, vous pourrez voir les listes de cadeaux de vos proches et ajouter des cadeaux à votre tour.

Il suffit de cliquer sur le lien, entrer votre mail (le même que celui sur lequel vous avez reçu ce mail) et vous serez connecté.

Si le lien ne s'affiche pas correctement, vous pouvez le copier-coller dans votre navigateur.
